# GemVision AI

## Overview

GemVision AI is a computer vision project aimed at automating the measurement and analysis of rough gemstones using image processing and AI.

The long-term goal is to:

* Detect rough gemstones from images
* Calculate real-world dimensions using a reference object
* Estimate optimal gemstone cuts
* Predict yield and material utilization
* Generate CAD-ready geometric designs

## Current Progress

### Completed

* Image loading using OpenCV
* Grayscale conversion
* Image thresholding
* Contour detection
* Object segmentation
* Bounding box generation

### In Progress

* Scale calculation using a reference coin
* Real-world dimension extraction

### Planned Features

* Top-view gemstone analysis
* Side-view thickness estimation
* Gemstone shape classification
* Cut optimization engine
* Yield prediction
* SVG/DXF CAD export
